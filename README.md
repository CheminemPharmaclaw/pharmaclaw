# PharmaClaw 🧪

AI-Powered Drug Discovery Pipeline — [pharmaclaw.com](https://pharmaclaw.com)

This repository contains the public website for PharmaClaw.

## About

PharmaClaw chains 9 AI agents into one drug discovery pipeline — chemistry, ADME, toxicology, synthesis, catalyst design, literature, IP expansion, market intelligence, and protein structure analysis.

Built on [OpenClaw](https://openclaw.ai) and published on [ClawHub](https://clawhub.com).
